## Estarei colocando comentarios no código para facilitar a edição caso nesessário.

### Para carregar as imagens no carrossel da home page:

- **Renomeie** as imagens como: image1, image2... OU img1,img2... OU apenas o numero 1, 2 ...
- Coloque as imagens renomeadas na pasta **images**
- O total de imagens aceitas no carrossel é 50.
- O carrossel carrega tres imagens por vez.
